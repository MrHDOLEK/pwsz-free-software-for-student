# Darmowe oprogramowanie i środowiska dla studentów
##### Aby uzyskać dostęp, do większości oprogramowania, należy podać uczelniany mail!!!
## Oprogramowanie i środowiska:
- [Wszystkie programy od Jet Brains np.phpstorm,inteli ij idea](https://www.jetbrains.com/community/education/#students)
- [Visual studio](https://visualstudio.microsoft.com/pl/students/)
- [Office 365](https://www.microsoft.com/pl-pl/education/products/office)
- [Windows 10](https://www.microsoft.com/pl-pl/education/products/windows)
- [Microsoft Azure (chmura)](https://azure.microsoft.com/pl-pl/)
- [AWS (chmura od amazonu)](https://aws.amazon.com/education/awseducate/)
- [Darmowy certyfikat + narzędzia od Cisco](https://www.cisco.com/c/en/us/solutions/industries/education.html)
- [GitHub Student Developer Pack](https://education.github.com/pack)
- [BitBucket](https://www.atlassian.com/software/views/bitbucket-academic-license.jsp)
- [Pakiet programów Autodesk np. AutoCad](https://www.autodesk.com/education/edu-software/overview?sorting=featured&page=2) tylko wybieramy inną uczelnie
